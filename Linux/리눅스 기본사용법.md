
#Linux Study

##명령어 정리 - Ubuntu 기준

-------------------------

- wget
	- web get의 약어
	- 웹 상의 파일을 현재폴더로 다운로드 받을 때 사용하는 명령어

	- 사용법 
		$ wget <옵션>... <URL>...
		ex) wget https://www.ebi.ac.uk/~zerbino/velvet/velvet_1.2.10.tgz

:		-o 옵션  다른이름으로 저장하고싶을 때
		ex)　wget –O target.zip  https://www.ebi.ac.uk/~zerbino/velvet/velvet_1.2.10.tgz
