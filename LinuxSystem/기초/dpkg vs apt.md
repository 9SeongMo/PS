
# dpkg vs apt
-----------------------------------------------

- ## dpkg
    - .deb 파일을 설치하고 싶어서 찾아보다
    - 리눅스에서는 .deb 파일이 윈도우에서 프로그램 설치 패키지 이라고 생각
    - dpkg
      - 설치 
        + ``` dpkg -i [패키지이름.deb]```
      - 삭제
        + ``` dpkg -P [패키지이름.deb] ```
        + apt 설치 패키지 에러날때 삭제해주니까 잘 됨
